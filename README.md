# mini-uc

mini-uc(**mini**-**U**til Components and React UI **C**omponents), 迷你的 React 工具组件&组件库。

如果你具有一些关于迷你组件的想法，欢迎提出 issue。mini-uc 致力于以轻量级的组件解决简单的交互。

### 安装

```
npm install mini-uc
```

```
yarn add mini-uc
```
